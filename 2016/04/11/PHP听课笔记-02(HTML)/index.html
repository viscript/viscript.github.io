PHP听课笔记-2

发表于 2016-04-11   |   分类于 PHP

本章主要写了HTML主体框架部分，主要针对框架内标签进行了描述和简介。
PHP代码编写部分。

代码分为三部分编写。

以下介绍大部分标签：

头标记

base标签

base标签：基底的网址标记。用于设定浏览器中文件的绝对路径。网页中文件只需要写下文件相对路径即可,这个路径就是base中指定下的路径。(当鼠标划过，韩剧i看到路径已经变成http://1/logo.gif)

link标签

link标签：设置外部文件的链接标记，用于确定本页面和其它文档之间的关系。

meta标签

meta标签：两种用法。
1.
2.

name用于在网页中加入一些关于网页的描述信息，网页的关键字，网页描述信息。
http-equiv：属性用于在HTML文档中模拟HTTP协议的响应消息。例如，告诉浏览器是否缓存页面，使用什么样的字符集显示网页内容。
meta中的name属性，不是自己定义的值。分为：

keywords 指定网页关键字
description 网页描述 （用|或者，隔开）
robots 有几个值：index noindex follow nofollow all是否允许搜索引擎搜索爬取。
author 作者
copyright 版本

meta标签中的http-equiv属性：

content-type 用于整个网页的类型和字符集
refresh 自动刷新。（例子中，5秒自动刷新，然后自动跳转到baidu.com）
expires 过期时间。例子中，0是代表禁止使用缓存页面。
windows-target 例子中，代表用户打开网页时，页面占领整个窗体。
pragma
page-enter和page-exit 特效，进入或者退出网页的时候，有进入或者退出特效。

主体标记

在body体内，放置网页所有内容。

test 页面中文本的颜色
bgcolor 背景颜色
background 背景图片
link 链接的颜色
alink 鼠标点到链接上的颜色
vlink 鼠标放到链接上的颜色
topmargin 顶部的距离
leftmargin 默认的左距离

但是这些都可以在CSS中来控制。只要是可以用样式来控制的，就不用HTML本身属性。

不仅是这些，还可以用id name class style等通用属性。是所有元素都可以使用的属性。

DTD文件

是文档类型定义，标签使用的语法格式。
<!DOCTYPE html PUBLIC “-//W3C//DTD XHTML 1.0 Transitionl//EN” “http://www.w3.org/TR/xhtml/DTD/xhtml1-transitionl.dtd">

html 整个页面是什么类型。
public 指明所依据的DTD文档是公用的，而不是个人所用。

  • ：ISO 标准的开头
    W3C 哪个组织发布的DTD文件
    version:DTD XHTML 1.0 DTD的版本是XHTML 1.0
    URL:transitionl 指的是DTD文件的地址。是从这个地址加载DTD。按照里边的类型显示页面。
