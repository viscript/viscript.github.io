PHP听课笔记-12-PHP

发表于 2016-04-14   |   分类于 PHP

PHP是一门服务器语言。

入门

这样处理之后，在页面查看源代码，能看到script内的源码，但是看不到PHP源码。因为用<?php?>规定之后，内部的源码经过PHP处理之后，将动态输出到页面，而不是在HTML直接输出。

而且，PHP标签可以插入到HTML中的任何位置。

指令分隔符

语句有两种：功能执行语句和结构定义语句。
功能执行语句后边，一定要加分号。但是结构定义语句一定不能加分号。
PS：与?>最近的一条语句，可以不加分号。

注释

1.// 单行注释
2./ / 多行注释
3.# 脚本注释
4./* / 文档注释

但是多行注释中，不能再包含多行注释。

程序中的空白

1.空格
2.tab
3.换行

另外注意

如果某条PHP脚本已经是程序最末尾，建议不要加 ?>，因为如果在别的脚本文件调用的时候，加了有可能会出错。
因为，很多函数都是属于头函数，在头函数前边不能有任何空白，但是原脚本文件的?>后边，可能会有很多空白，因此导致了出错。
