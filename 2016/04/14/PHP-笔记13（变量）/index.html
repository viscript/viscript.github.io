PHP-笔记（变量）

发表于 2016-04-14   |   分类于 PHP

变量是用于临时存储值的容器。

定义变量

如果在用到数据时，需要用到多次，最好声明变量。
定义方式：$变量名=值
–>PHP是弱类型语言。
强类型语言：C,java等，声明变量一定要先指定类型，这样只能存储指定类型的内容。
PHP变量的类型由存储的值决定。

与变量有关的函数

1.isset 判断是否存在
2.unset 释放定义的字符

变量的命名

1.变量前一定要使用$符。而且声明和使用都要有$符号。
2.不能以数字开头。###
3.不能使用PHP或者其它语言的运算符号。
4.PHP可以使用系统关键字作为变量名。
5.PHP变量区分大小写。（只有变量和常量区分大小写，其它不区分）
6.变量名称最好是有意义。
7.引用赋值：$two=&$one ，是把one作一个别名，叫做two，但是不能把值的地址赋予，只能是变量名的地址赋予。

变量的类型

四种标量

1.整型 int integer
2.布尔型 bool boolean
3.浮点型 float double real
4.字符串 string

两种复合类型

1.数组 存多个变量 array
2.对象 存变量和函数 object

两种特殊类型

1.资源类型 resource
2.空类型 null

变量声明

1.单双引号都可以用来声明字符串。
2.声明的字符串没有长度限制。
3.双引号的字符串中，既可以直接解析变量，也可以直接使用转义字符。
4.单引号的字符串中，不可以解析变量和转义字符，但可以转义单引号本身。
5.双引号中不能使用双引号，单引号中不能使用单引号。
